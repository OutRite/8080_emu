# Space Invaders Emulator
 A Space Invaders arcade emulator that doesn't work.  
It also contains some (small) CP/M HLE code to use with cpudiag.
## Controls
1p left: left arrow key  
1p right: right arrow key  
insert coin: q  
1p start: a  
1p shoot: z  
2p left: j  
2p right: l  
2p start: s  
2p shoot: m  
tilt: t
